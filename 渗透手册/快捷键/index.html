用于记录平时见到的一些快捷键
通常情况下，windows的快捷键也能够在其他的软件中使用。

有些笔记本键盘尺寸较小，没有右侧的数字键盘区
有些外设键盘一个键能搞定的，笔记本键盘可能需要使用fn +按键

Windows

打开运行窗口

组合键： Windows + R

复制

组合键：Ctrl + C

介绍： 需要先选中文本内容或文件

粘贴

组合键：Ctrl + V

组合键： Shift + Insert

介绍： 需要提前复制或剪切内容，才能进行黏贴

粘贴板

组合键：Windows + V

剪切

组合键：Ctrl + X

介绍： 需要先选中文本内容或文件

多选

组合键：Shift + 左键

组合键：Ctrl + 左键

介绍： 使用时需要提前选中一个文件，然后使用组合键点击另一个文件。

全选

组合键： Ctrl + A

软件用户指南

快捷键：F1

大小写转换

快捷键：CapsLk

快捷键：CapsLock

介绍： 有的笔记本是CapsLk，外设键盘通常是CapsLock

切换桌面

组合键：Windows + D

组合键：Windows + M

输入法切换

组合键：Ctrl + Shift

介绍： 之前看有人切换输入法时，按下组合键后用力往左搓按键，大可不必，太用力按键盘还容易导致按键损坏。

窗口切换

组合键：Alt + Tab

介绍： 先按住Alt键，再按下Tab键可以切换到上一个窗口，如果有多个打开的窗口时，可以不松开Alt键，使用Tab键

笔记本键盘灯

组合键： Fn + Space

介绍： Space即空格键，我自己的键盘灯是有两个状态，第一个状态灯微亮，第二个状态是全亮，再按下组合键就是关闭灯光。

刷新桌面

快捷键：F5

介绍： 在使用快捷键的时候，需要先把界面切换到桌面

回退上一操作

组合键： Ctrl + Z

介绍： 返回上一个操作

回退至上一目录

组合键： Alt + ↑

任务管理器

组合键：Ctrl + Alt + Delete

组合键：Ctrl + Shift + Esc

组合键： Ctrl + Alt + .

介绍： 点为数字键盘的小数点

中英文切换

快捷键：Shift

文件重命名

快捷键：F2

介绍： 需要先选中一个文件，再使用快捷键才能重命名

电脑锁屏

组合键：Windows + L

缩放界面

组合键：Ctrl + 滚轮

介绍： 需要按住Ctrl键，再使用滚轮放大或缩小界面，适用于软件、网页、编辑器等能够输入文字、或是查看图片的地方。

窗口最小化

组合键： Windows + ↓

介绍：
如果你的窗口是处于最大化状态
第一次使用组合键时将会使窗口取消最大化
第二次使用组合键后会将窗口最小化到任务栏中
第三次使用组合键后会取消在任务栏中对该窗口的选择。

窗口最大化

组合键： Windows + ↑
如果你的窗口是处于最小化状态，第一次使用组合键时将会从任务栏中调出窗口，第二次使用组合键后会将窗口最大化。

表情符号

组合键： Windows + : ;
组合键： Windows + > .

关闭窗口

组合键： Ctrl + W

文件资源管理器

组合键： Windows + E

Windows设置

组合键： Windwos + I

放大镜

组合键： Windows + 加(+)，放大
组合键： Windows + 减(-)，缩小
组合键： Windows + Esc ，关闭

关闭程序

组合键： Alt + F4

介绍： 使用该组合键相当于点击了程序窗口右上角的x

竖排选中

组合键： ALT + 左键

介绍： 按住ALT和鼠标左键后移动鼠标选中所需内容

选中托盘栏

组合键： Windows + B

打开当前窗口菜单

组合键： Alt + 空格

移动滑块

上移快捷键： PageUp

下移快捷键： PageDown

顶部快捷键： Home

底部快捷键： End

介绍： 在笔记本的键盘上可能会缩写放置到右侧数字区，此时需要先将NumberBlock设置关闭，关闭数字的输入之后才能使用功能键PageUp、PageDown、Home和End等

关闭当前浏览器标签

组合键： Ctrl + W

刷新当前浏览器页面

组合键： Ctrl + R

换行

快捷键： 回车
组合键： CTRL + 回车
介绍： 有的时候，聊天框的回车键会被用来直接发送消息，此时就可以使用CTRL + 回车来直接换行

隐藏菜单栏

快捷键： ALT

Macbook

关闭当前程序

快捷键：Command + Q
